📈 K線（蠟燭圖）是什麼？

你在交易所看到的圖表，每一根「蠟燭」就是一根 K線，代表一段時間內的價格變化。

一根K線包含4個資訊：

🔴 紅色K線（陽線）
收盤價 > 開盤價
代表這段時間上漲
🟢 綠色K線（陰線）
收盤價 < 開盤價
代表這段時間下跌
💡 台灣/中國習慣：紅色=漲、綠色=跌（和歐美相反）。大多數亞洲交易所都是這個顯示方式。

⏱️ 時間週期是什麼？

每根K線代表多長時間，取決於你選的週期：

週期 每根K線代表 適合誰
1分鐘1分鐘的價格極短線、scalping
15分鐘15分鐘的價格日內短線
1小時1小時的價格⭐ 建議新手從這裡開始
4小時4小時的價格波段交易
日線(1D)1天的價格中長線投資
💡 圖片中建議「1小時及以上週期，周期越大可靠性越高」。新手建議從1小時或4小時開始練習觀察。

🎯 布林帶（Bollinger Bands / BOLL）

布林帶是由3條線組成，包圍著K線走動，像一個價格通道。

布林帶示意圖

3條線各代表什麼？

🟠 上軌（Upper Band） — 壓力線
= 中軌 + 2倍標準差。價格靠近這裡代表可能偏高，注意回落。
🔵 中軌（Middle Band） — 均線（最重要！）
= 20日均線（MA20）。這是趨勢方向的核心。中軌向上＝多頭，向下＝空頭，走平＝震盪。
🔵 下軌（Lower Band） — 支撐線
= 中軌 − 2倍標準差。價格靠近這裡代表可能偏低，注意反彈。

🔑 布林帶的4個關鍵判斷

🟠

通道開口擴張 → 趨勢加速中

上軌和下軌距離越來越大 = 行情在快速移動。多頭時配合加倉，空頭時持有空單。

🔵

布林帶縮口 → 不要進場！

3條線靠攏在一起 = 橫盤震盪，方向不明。這時候進場容易被打止損，觀望為主。

🔴

價格突破中軌（向上）→ 多頭啟動信號

價格從中軌下方穿越到中軌上方，中軌開始走平或向上 = 買入機會開始出現。

🟢

價格跌破中軌（向下）→ 空頭啟動信號

價格從中軌上方穿越到中軌下方，中軌開始走平或向下 = 賣出/做空機會開始出現。

⚡ MACD 是什麼？

MACD（移動平均匯聚/發散指標）是用來衡量趨勢的動能強弱，判斷行情是在加速還是減速。

它由3個部分組成：

MACD 指標示意圖（出現在K線圖下方）

🔍 MACD 3個組成部分詳解

🟠 DIF（快線/白線）
= 12日均線 − 26日均線。反應較快，先行感應趨勢變化。
交易所上常顯示為白色或橙色
🔵 DEA（慢線/信號線）
= DIF的9日均線。反應較慢，代表趨勢的確認。
交易所上常顯示為藍色或黃色
🟥🟩 MACD柱（Histogram）
= DIF − DEA。
• 柱子在零軸上方且紅色變大 = 多頭動能增強
• 柱子在零軸下方且綠色變大 = 空頭動能增強

最重要的3個信號：

信號發生了什麼意義
金叉 DIF（快線）從下方穿越DEA（慢線）向上 買入信號，多頭啟動
死叉 DIF（快線）從上方穿越DEA（慢線）向下 賣出信號，空頭啟動
零軸位置 金叉發生在零軸上方 or 下方 上方金叉更強，下方金叉更弱

🔗 布林帶 + MACD 如何搭配？

單獨用一個指標容易有假信號。兩個指標同時出現信號才叫「共振」，可靠性更高。

共振原理：
✅ 布林帶說「突破中軌向上」 + MACD說「零軸附近金叉」
= 兩個指標都同意上漲 → 強烈買入信號

❌ 只有布林帶突破中軌，MACD還在死叉狀態
= 信號不夠強，等待確認或觀望

🔧 如何在任何交易所開啟 BOLL + MACD

無論你用的是幣安、OKX（歐易）、富途、永豐、國泰君安，步驟基本上都一樣：

1

進入K線圖（圖表頁面）

點擊任何一個股票或幣種，找到「圖表」或「K線」按鈕。通常在交易頁面頂部。

2

找到「指標」按鈕

在圖表工具列找到「指標」、「Indicators」、「技術指標」或一個類似「f(x)」的按鈕，點擊它。

3

搜尋並加入「BOLL」（布林帶）

在搜尋框輸入「BOLL」或「Bollinger」，點擊加入。圖表上會出現3條線（上軌、中軌、下軌）。

4

搜尋並加入「MACD」

再次搜尋「MACD」，點擊加入。圖表下方會出現新的區域，顯示柱子和快慢線。

5

選擇時間週期

在圖表頂部點選時間週期。建議新手先選「1H（1小時）」或「4H（4小時）」觀察。

💡 預設參數就可以用！
BOLL 預設: 週期20, 標準差2
MACD 預設: 快線12, 慢線26, 信號線9
不需要修改任何數值，用預設值即可。

📋 8種 BOLL+MACD 共振信號總覽

以下是你圖片中完整的「布林帶+MACD期貨共振戰法」，我把每一種都用簡單的話解釋：

💡 信號分3大類：做多（買入）、做空（賣出）、特殊情況

🔴 多頭信號（準備做多/買入）

①

多頭啟動

趨勢初段

布林帶條件：

價格突破中軌向上，中軌走平或向上

MACD條件：

零軸附近或上方金叉，紅柱開始放大

操作信號：

做多（買入）

止損設在：

中軌下方 或 那根K線的最低點

💬 用大白話說：價格剛從中線下面穿上去，同時MACD也剛金叉（快線穿過慢線往上），表示上漲趨勢剛剛開始。現在買進，把止損放在中線下面。
②

多頭回調

上升中繼

布林帶條件：

價格回踩中軌但沒跌破，中軌仍向上

MACD條件：

零軸上方二次金叉 或 綠柱縮短

操作信號：

加多（追加買入）

止損設在：

中軌下方

💬 用大白話說：上漲途中，價格往下跌到中線附近「補了口氣」，但沒跌破中線，MACD也在零軸上方再次金叉。這是上漲中途的補倉機會。
③

多頭加速

主升浪

布林帶條件：

價格沿上軌運行，通道開口擴張

MACD條件：

紅柱持續放大，快慢線發散（距離越來越大）

操作信號：

持有 / 加倉

止損移到：

中軌（跟著移動，保護獲利）

💬 用大白話說：行情正在飆漲！價格貼著上軌走，布林帶開口越來越大，MACD紅柱越來越高。這時候繼續持有，止損往上移到中線，順勢賺最多。

🟢 空頭信號（準備做空/賣出）

💡 如果你只買現貨（不做空），空頭信號代表「不要持有」或「把手上的貨賣掉」。

④

空頭啟動

趨勢初段

布林帶條件：

價格跌破中軌，中軌走平或向下

MACD條件：

零軸附近或下方死叉，綠柱開始放大

操作信號：

做空 / 賣出

止損設在：

中軌上方 或 那根K線的最高點

💬 大白話：價格剛從中線上面穿下去，MACD死叉（快線穿過慢線往下）。下跌趨勢剛開始，現貨持有者應賣出。
⑤

空頭反彈

下降中繼

布林帶條件：

價格反彈到中軌附近但穿不過，中軌仍向下

MACD條件：

零軸下方二次死叉 或 紅柱縮短

操作信號：

加空

止損設在：

中軌上方

💬 大白話：下跌途中反彈到中線，但衝不過去，MACD在零軸下方再次死叉。這是下跌中繼，逢高賣出的機會。
⑥

空頭加速

主跌浪

布林帶條件：

價格沿下軌運行，通道開口擴張

MACD條件：

綠柱持續放大，快慢線發散

操作信號：

持有空單 / 加倉

止損移到：

中軌（往下移動）

💬 大白話：行情在飆跌！價格貼著下軌，MACD綠柱越來越大。繼續持有空單，止損移至中線，空單趁機加碼。

⚠️ 特殊情況信號

⑦A

反轉預警 — 頂背離（上漲末端）

布林帶條件：

價格衝出上軌後回落

MACD條件：

頂背離（價格創新高，但MACD峰值反而降低）

操作信號：

多單止盈 或 輕倉試空

止損設在：

背離高點上方

💬 大白話：價格雖然創了新高，但MACD的高點卻比上一次更低，代表上漲動能在衰竭。準備反轉的訊號！先把多單利潤落袋，或少量試空。
⑦B

反轉預警 — 底背離（下跌末端）

布林帶條件：

價格衝出下軌後回升

MACD條件：

底背離（價格創新低，但MACD谷值反而升高）

操作信號：

空單止盈 或 輕倉試多

止損設在：

背離低點下方

💬 大白話：價格雖然創了新低，但MACD的低點卻比上一次更高，代表下跌動能在衰竭。準備反彈的訊號！先把空單利潤落袋，或少量試多。
⑧

無效信號（震盪行情）

觀望

布林帶條件：

布林帶嚴重縮口，3條線靠在一起走平

MACD條件：

MACD在零軸附近反覆交叉，多空不明

操作信號：

觀望，不進場

適合場景：

橫盤震盪行情

💬 大白話：布林帶的3條線縮成一團，MACD在零軸上下亂跑。這種情況下進場容易被假信號騙，最好的選擇就是什麼都不做，等到方向明確再說。

🔄 背離（Divergence）核心操作要點

背離是整個策略中最精準的反轉信號，但也最需要耐心確認。圖片中完整列出了操作要點：

什麼是「背離」？

📊 核心操作要點對照表

🔴 頂背離（賣出信號）

背離確認條件：

價格第2高點 > 第1高點
但 DIF 第2高點 < 第1高點

最佳週期：

⭐ 1小時以上（週期越大越可靠）

入場確認信號：

出現吞沒陰線、黃昏星等K線反轉型態

倉位管理：

初次試空倉位減半（這叫「左側交易」，較保守）

失效條件：

價格繼續上漲 並帶動MACD創新高 → 背離信號無效，止損離場

🟢 底背離（買入信號）

背離確認條件：

價格第2低點 < 第1低點
但 DIF 第2低點 > 第1低點

最佳週期：

⭐ 1小時以上（週期越大越可靠）

入場確認信號：

出現吞沒陽線、晨星等K線反轉型態

倉位管理：

初次試多倉位減半（「左側交易」，較保守）

失效條件：

價格繼續下跌 並帶動MACD創新低 → 背離信號無效，止損離場

💡 新手學背離的3個重點

1

先用1小時或4小時圖練習

背離在越大的時間框架越可靠。1分鐘圖的背離幾乎沒用，1小時以上才有參考價值。

2

背離出現≠馬上進場

看到背離後，要等K線出現「確認信號」（如大陽線/大陰線吞沒前一根），再決定是否進場。

3

一定要設止損

背離失效率不低。如果價格繼續突破創新高/新低並且MACD也創新高/新低，代表背離失效，必須止損。

⚠️ 風險提示：技術分析是輔助工具，不是保證獲利的魔法。任何交易都應設置嚴格的止損，只用自己能承受虧損的資金操作。

📌 快速查詢卡（Real-time Reference）

把這頁截圖或收藏！開交易所看到行情時，對照下面快速判斷：

🔴 做多信號

🚀 多頭啟動 價格穿越中軌向上 + MACD零軸金叉 + 紅柱放大 買入
📉 多頭回調 回踩中軌不破 + 零軸上方二次金叉 加多
⚡ 多頭加速 貼上軌走 + 通道擴張 + 紅柱持續放大 持有

🟢 做空信號

📉 空頭啟動 價格跌破中軌 + MACD零軸死叉 + 綠柱放大 賣出
📈 空頭反彈 反彈到中軌過不去 + 零軸下方二次死叉 加空
💥 空頭加速 貼下軌走 + 通道擴張 + 綠柱持續放大 持有

⚠️ 特殊情況

🔔 頂背離 價格新高 + MACD峰值更低 + 衝出上軌回落 止多/試空
🔔 底背離 價格新低 + MACD谷值更高 + 衝出下軌回升 止空/試多
🚫 無效信號 布林帶縮口走平 + MACD零軸反覆交叉 觀望

🎯 快速判斷流程（5步驟）

1

看布林帶縮口了嗎？

如果是 → 觀望，什麼都不做。如果否 → 繼續第2步。

2

看中軌方向

中軌向上 = 偏多思路；中軌向下 = 偏空思路；中軌走平 = 觀望。

3

確認MACD位置

DIF/DEA在零軸上方 = 多頭；在下方 = 空頭；紅柱放大 = 多頭加速。

4

找共振信號

布林帶和MACD都說同一件事（都多頭或都空頭）才進場，對應上方8種信號之一。

5

設止損再進場

根據你的信號找到止損位（中軌下方/上方，或前一根K線高低點），確認好才下單。

  5. 先「紙上模擬交易」（不用真錢），觀察信號後價格的走向
